One-line: Circular peer avatar with initials fallback and an optional presence dot.

```jsx
<Avatar name="Willow Adeyemi" status="relaying" size="lg" />
<Avatar src="/p.jpg" name="Rafi" status="online" />
```
Status: `online` (moss), `relaying` (terracotta), `offline` (clay). Initials color is derived deterministically from the name.
