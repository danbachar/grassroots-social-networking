One-line: Square icon-only button for toolbars and compact actions; always pass `label` for accessibility.

```jsx
<IconButton label="Attach" variant="ghost"><PaperclipIcon/></IconButton>
<IconButton label="Send" variant="solid"><SendIcon/></IconButton>
```

Variants: `ghost` (default), `soft`, `solid`, `outline`. Sizes `sm|md|lg`. Presses scale to 0.92.
