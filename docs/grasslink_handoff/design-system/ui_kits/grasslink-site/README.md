# grasslink site — UI kit

Recreation of the grasslink marketing landing page.

## Sections (`sections.jsx`)
- **Nav** — sticky, blurred, wordmark + links + Join CTA.
- **Hero** — headline, subcopy, dual CTAs, peer-avatar social proof, and a live moss "mesh card" preview with a floating hop-count chip.
- **HowItWorks** — three Cards ("Join / Relay / Reach the world") on a sunken band.
- **Stats** — four display-type figures (peers, messages, meshes, towers).
- **CTA** — terracotta full-bleed panel.
- **Footer** — dark clay, link columns, wordmark.

Composed from design-system components (Button, Badge, Card, SignalMeter, Avatar). Icons via Lucide (CDN). All copy follows the brand voice — warm, plain-spoken, collective.
